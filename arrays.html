📚 Learn JavaScript Arrays Step by Step

Arrays are lists that store many values in one variable. This page helps you practice array basics and common array methods.

1️⃣ Create and Access Arrays

let colors = ["red", "green", "blue"];
let firstColor = colors[0];      // "red"
let secondColor = colors[1];     // "green"
let lastColor = colors[colors.length - 1];

2️⃣ Add and Remove Items

let numbers = [1, 2, 3];
numbers.push(4);     // add to end
numbers.unshift(0);  // add to start
numbers.pop();       // remove last item
numbers.shift();     // remove first item

3️⃣ Array Length and Index

let items = ["apple", "banana", "cherry"];
let count = items.length;          // 3
let index = items.indexOf("banana");  // 1
let hasApple = items.includes("apple"); // true

4️⃣ Loop Through an Array

let letters = ["a", "b", "c"];
for (let i = 0; i < letters.length; i++) {
    console.log(letters[i]);
}

5️⃣ Array Methods: map, filter, and find

let scores = [10, 20, 30, 40];
let doubled = scores.map(n => n * 2);
let highScores = scores.filter(n => n >= 30);
let firstHigh = scores.find(n => n >= 30);

6️⃣ Modify Arrays with splice

let days = ["Mon", "Tue", "Thu", "Fri"];
days.splice(2, 0, "Wed");      // insert at index 2
days.splice(4, 1);              // remove "Fri"

7️⃣ Combine Arrays

let a = [1, 2];
let b = [3, 4];
let combined = a.concat(b);
let alsoCombined = [...a, ...b];

8️⃣ Nested Arrays

let matrix = [
    [1, 2],
    [3, 4]
];
let value = matrix[1][0];  // 3

9️⃣ Practice Challenge

// Create a function that returns even numbers from an array
function getEvenNumbers(numbers) {
    return numbers.filter(n => n % 2 === 0);
}

💡 Key Array Concepts

  • An array stores ordered values inside square brackets []
  • Use an index to get items: array[0] is the first item
  • push/pop change the end; shift/unshift change the start
  • map, filter, and find are powerful ways to transform arrays
  • splice can add or remove items at any position
