📚 Learn Loops and Mapping Step by Step

1️⃣ FOR LOOPS - Repeat Code Multiple Times

What is a for loop? A way to repeat code a specific number of times

Step 1.1: Basic For Loop

for (let i = 0; i < 5; i++) {
    console.log("Count: " + i);
}

Step 1.2: For Loop with Arrays

let fruits = ["apple", "banana", "cherry"];
for (let i = 0; i < fruits.length; i++) {
    console.log("Fruit " + (i + 1) + ": " + fruits[i]);
}

Step 1.3: For Loop with Calculations

let sum = 0;
for (let i = 1; i <= 10; i++) {
    sum = sum + i;
}
console.log("Sum of 1-10: " + sum);

2️⃣ WHILE LOOPS - Repeat Until Condition is False

What is a while loop? Keeps running as long as a condition is true

Step 2.1: Basic While Loop

let count = 0;
while (count < 5) {
    console.log("Count: " + count);
    count = count + 1;
}

Step 2.2: While Loop with User Input Simulation

let number = 1;
while (number <= 10) {
    console.log(number + " x 2 = " + (number * 2));
    number++;
}

3️⃣ FOR...OF LOOPS - Loop Through Arrays Easily

What is for...of? A simpler way to loop through array elements

Step 3.1: For...of with Arrays

let colors = ["red", "green", "blue"];
for (let color of colors) {
    console.log("Color: " + color);
}

Step 3.2: For...of with Strings

let word = "HELLO";
for (let letter of word) {
    console.log("Letter: " + letter);
}

4️⃣ ARRAY MAPPING METHODS - Transform Arrays

What are array methods? Built-in functions that work with arrays

Step 4.1: forEach - Do Something with Each Item

let numbers = [1, 2, 3, 4, 5];
numbers.forEach(function(num) {
    console.log(num * 2);
});

Step 4.2: map - Transform Each Item

let prices = [10, 20, 30];
let discountedPrices = prices.map(function(price) {
    return price * 0.8; // 20% discount
});

Step 4.3: filter - Keep Only Certain Items

let scores = [85, 92, 78, 95, 88];
let highScores = scores.filter(function(score) {
    return score >= 90;
});

Step 4.4: reduce - Combine All Items

let numbers = [1, 2, 3, 4, 5];
let total = numbers.reduce(function(sum, num) {
    return sum + num;
}, 0);

5️⃣ PRACTICAL EXAMPLES - Real World Use

Step 5.1: Shopping Cart Total

let cart = [
    {name: "Apple", price: 2.50, quantity: 3},
    {name: "Banana", price: 1.20, quantity: 2},
    {name: "Orange", price: 3.00, quantity: 1}
];

let total = cart.reduce(function(sum, item) {
    return sum + (item.price * item.quantity);
}, 0);

Step 5.2: Student Grade Calculator

let students = [
    {name: "Ali", grade: 85},
    {name: "Sara", grade: 92},
    {name: "Ahmed", grade: 78}
];

let passedStudents = students.filter(function(student) {
    return student.grade >= 80;
});

let averageGrade = students.reduce(function(sum, student) {
    return sum + student.grade;
}, 0) / students.length;

Step 5.3: Temperature Converter

let celsiusTemps = [0, 10, 20, 30, 40];
let fahrenheitTemps = celsiusTemps.map(function(celsius) {
    return (celsius * 9/5) + 32;
});

💡 Key Takeaways

  • For loops: Use when you know how many times to repeat
  • While loops: Use when you don't know how many times (condition-based)
  • For...of: Simple way to loop through arrays and strings
  • forEach: Do something with each item (no return value)
  • map: Transform each item into something new
  • filter: Keep only items that match a condition
  • reduce: Combine all items into a single value
