5️⃣ EVENT HANDLING - Responding to User Actions

Step 5.1: addEventListener

let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("Button clicked!");
});

Step 5.2: Multiple Event Types

element.addEventListener("mouseover", function() {
    element.style.backgroundColor = "lightblue";
});

element.addEventListener("mouseout", function() {
    element.style.backgroundColor = "white";
});

    💡 Key Takeaways

    • DOM: Document Object Model - HTML as a tree structure
    • Selectors: getElementById, querySelector, getElementsByClassName
    • Content: innerHTML for HTML, textContent for text
    • Styles: element.style or classList for CSS changes
    • Events: addEventListener to respond to user actions
    • Creation: createElement and appendChild to add new elements
