📚 Learn Objects and Classes Step by Step

1️⃣ OBJECTS - Data Containers

What is an object? A collection of properties and methods that represent real-world things

Step 1.1: Creating Objects

let person = {
    name: "Hamad",
    age: 25,
    city: "Karachi"
};

Step 1.2: Accessing Object Properties

let person = {
    name: "Hamad",
    age: 25,
    city: "Karachi"
};

let name = person.name;        // Dot notation
let age = person["age"];       // Bracket notation

Step 1.3: Adding and Modifying Properties

let car = {
    brand: "Toyota",
    model: "Corolla"
};

car.year = 2020;              // Add new property
car.model = "Camry";           // Modify existing property

Step 1.4: Object Methods

let calculator = {
    add: function(a, b) {
        return a + b;
    },
    multiply: function(a, b) {
        return a * b;
    }
};

2️⃣ CLASSES - Object Blueprints

What is a class? A template for creating objects with the same structure

Step 2.1: Creating a Class

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        return "Hello, I'm " + this.name;
    }
}

Step 2.2: Creating Objects from Classes

class Car {
    constructor(brand, model, year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
    }

    getInfo() {
        return this.year + " " + this.brand + " " + this.model;
    }
}

let myCar = new Car("Honda", "Civic", 2022);

Step 2.3: Class Inheritance

class Animal {
    constructor(name) {
        this.name = name;
    }

    speak() {
        return this.name + " makes a sound";
    }
}

class Dog extends Animal {
    speak() {
        return this.name + " barks";
    }
}

3️⃣ PRACTICAL EXAMPLES

Step 3.1: Student Management System

class Student {
    constructor(name, grades) {
        this.name = name;
        this.grades = grades;
    }

    getAverage() {
        let sum = this.grades.reduce((a, b) => a + b, 0);
        return sum / this.grades.length;
    }

    getGrade() {
        let avg = this.getAverage();
        if (avg >= 90) return "A";
        if (avg >= 80) return "B";
        if (avg >= 70) return "C";
        return "F";
    }
}

Step 3.2: Bank Account System

class BankAccount {
    constructor(owner, balance = 0) {
        this.owner = owner;
        this.balance = balance;
    }

    deposit(amount) {
        this.balance += amount;
        return "Deposited $" + amount + ". New balance: $" + this.balance;
    }

    withdraw(amount) {
        if (amount > this.balance) {
            return "Insufficient funds";
        }
        this.balance -= amount;
        return "Withdrew $" + amount + ". New balance: $" + this.balance;
    }
}

💡 Key Takeaways

  • Objects: Store data as key-value pairs
  • Properties: Values stored in objects
  • Methods: Functions stored in objects
  • Classes: Blueprints for creating objects
  • Constructor: Special method that creates objects
  • Inheritance: Classes can extend other classes
